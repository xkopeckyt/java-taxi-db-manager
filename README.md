# PV168 Project: Share Car Rider

This application is developed for storing information about rides (price, currency, distance, date and time, passengers count and their categories).

You can filter rides by their parameters, view global and filtered statistics, create, edit, delete and assign categories to rides, edit individual rides and type in technical license validity date, so you could be informed about its invalidity.
